## 🎨 주요 기능 스크린샷 시나리오

### 1️⃣ 파트너 회원가입
1. 홈페이지 접속
2. "파트너 등록하기" 클릭
3. 기본 정보 입력
4. 상점 정보 입력 (자동 URL 생성)
5. 6개 라이브 플랫폼 중 선택
6. 회원가입 완료

### 2️⃣ 파트너 대시보드
1. 파트너 로그인
2. 실시간 통계 확인
   - 오늘 매출: ₩0
   - 총 매출: ₩132,000
   - 정산 대기: ₩39,600
3. 최근 주문 목록 확인
4. 주문별 수익 확인

### 3️⃣ 관리자 - 제품 관리
1. 관리자 로그인
2. 제품 관리 메뉴 클릭
3. 제품 카드 그리드 확인
4. 제품 상태 토글 (판매중/중지)
5. 판매 통계 확인
   - 주문 건수
   - 파트너 사용 수
   - 재고 수량

### 4️⃣ 관리자 - 주문 관리
1. 주문 관리 메뉴 클릭
2. 주문 목록 확인
3. 주문 상태별 필터링
4. 주문 상태 변경
   - "대기중" → "주문 확인" 버튼
   - "확인됨" → "배송 시작" 버튼
   - "배송중" → "배송 완료" 버튼
5. 고객/파트너 정보 확인
6. 수익 분배 확인
   - 주문 금액
   - 파트너 수익 (30%)
   - 플랫폼 수익 (70%)

## 💰 수익 분배 시스템

### 자동 계산 로직
```typescript
주문 금액: ₩100,000

파트너 수익 = ₩100,000 × 30% = ₩30,000
플랫폼 수익 = ₩100,000 × 70% = ₩70,000
```

### 정산 프로세스
1. ✅ 주문 생성 → 수익 자동 분배 계산
2. ✅ 주문 확인 → 정산 대기 상태
3. 🚧 정산 요청 (구현 예정)
4. 🚧 정산 완료 → 파트너에게 지급 (구현 예정)
